Illumination of Light Using Bluetooth

"Illumination of Light Using Bluetooth" is a Home Automation project I developed. It enables users to control home lighting via Bluetooth connectivity. The project enhances convenience by allowing users to remotely manage lighting conditions, demonstrating the integration of Bluetooth technology for efficient and accessible home automation.
